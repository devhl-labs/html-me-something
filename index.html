CocApi

CocApi is a wrapper for the Clash of Clans Api. It is written in .Net Standard. This library is able to poll the clash API and fire events when a change is seen.

You can configure this library to watch the end points that you care about. You also have control over how long it considers a cached response valid for.

Themify

Themify is a Google Chrome extension. It will apply dark theme to your search results. The theme is customizable.

There are four built-in CSS themes, and each one is fully customizable. If you don't like the default colors, or if you want to improve what's already there, just type the CSS changes in the popup window!
